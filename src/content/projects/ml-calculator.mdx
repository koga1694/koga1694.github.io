## 자주 쓰는 계산들

### GPU 메모리 계산

모델 파라미터 수에서 필요한 메모리를 계산할 때:

- **FP32**: 파라미터 수 × 4 bytes
- **FP16**: 파라미터 수 × 2 bytes
- **학습 시**: 추가로 gradients + optimizer states 필요

예: 7B 모델 (FP16)
```
7,000,000,000 × 2 = 14,000,000,000 bytes = 약 13GB
```

### 배치 크기와 학습 시간

총 학습 스텝 수:
```
총 스텝 = (데이터셋 크기 / 배치 크기) × 에폭 수
```

### 토큰 수 추정

영문 기준 대략적인 추정:
```
토큰 수 ≈ 단어 수 × 1.3
토큰 수 ≈ 문자 수 / 4
```
