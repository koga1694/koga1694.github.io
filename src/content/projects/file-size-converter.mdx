## 기능

- **실시간 변환**: 값을 입력하면 즉시 모든 단위로 변환됩니다
- **바이너리 기준**: 1 KB = 1024 Bytes (IEC 표준)
- **넓은 범위**: Bytes부터 PB까지 지원

## 구현 코드

```tsx
const units = ['Bytes', 'KB', 'MB', 'GB', 'TB', 'PB'];
const multipliers = [1, 1024, 1024 ** 2, 1024 ** 3, 1024 ** 4, 1024 ** 5];

const convert = (value: number, fromUnit: string) => {
  const fromIndex = units.indexOf(fromUnit);
  const bytesValue = value * multipliers[fromIndex];

  return units.map((unit, index) => ({
    unit,
    value: bytesValue / multipliers[index],
  }));
};
```

## 활용 사례

1. **S3 비용 계산**: 저장 용량을 GB/TB로 변환하여 월 비용 추정
2. **네트워크 대역폭**: 전송 속도와 파일 크기 비교
3. **Docker 이미지 최적화**: 레이어별 크기 분석
