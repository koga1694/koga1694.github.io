> 개발자가 Claude Code CLI에서 할 수 있는 모든 것을, 코딩을 전혀 모르는 사람도 Discord에서 대화하듯 할 수 있게 해주는 도구.

## 한 줄 설명

Discord에서 "포트폴리오 사이트 만들어줘"라고 말하면, AI가 실제 코드를 짜고 배포까지 해준다.

사용자는 코딩을 몰라도 된다. 개발 환경 설정도, Node.js가 뭔지도 몰라도 된다. 그냥 Discord에서 말만 하면 된다.

---

## 왜 만들었나

Claude Code 같은 AI 코딩 도구들이 점점 강력해지고 있는데, 정작 "비개발자"는 여전히 접근하기 어렵다. CLI를 써야 하고, 터미널이 뭔지 알아야 하고, API 키를 어디서 발급하는지도 알아야 한다.

DoItYourself는 그 장벽을 없애려는 시도다. **AI 코딩 에이전트의 기능은 그대로**, 진입 장벽만 없애는 것.

---

## 핵심 원칙

**1. 추상화가 아닌 번역**

개발 과정을 숨기지 않는다. 개발자가 CLI에서 보는 것들 — `npm install`, 파일 편집, 빌드 에러 — 을 비개발자가 이해할 수 있는 말로 실시간 번역해서 보여준다.

"필요한 도구를 설치하고 있어요..." (= `npm install` 실행 중)  
"파일을 수정하고 있어요..." (= 코드 파일 편집 중)  
"문제가 있어서 고치고 있어요... 잠깐만요!" (= 에러 자동 수정 중)

**2. 동일한 파워**

Claude Code CLI를 쓰는 개발자와 **동일한 수준**의 기능을 제공한다. 파일 편집, Bash 명령 실행, 패키지 설치, 테스트, 빌드, 배포까지 모두. 기능을 제한하거나 단순화하지 않는다.

**3. 극도의 접근성**

더블클릭 설치 → 10분 이내 셋업 → 바로 사용.

Docker가 뭔지, Node.js가 뭔지, 클라우드가 뭔지 몰라도 된다.

**4. 셀프호스팅**

각 사용자가 자신의 AI 계정과 Discord 봇을 직접 쓴다. 데이터는 본인 PC에만 저장된다. 중간에 서버가 없다.

**5. 멀티 AI 엔진**

Gemini (무료), Claude, OpenAI Codex 중 선택 가능. 이미 구독 중인 서비스를 그대로 쓸 수 있다.
